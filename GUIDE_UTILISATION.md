# 📖 Guide d'Utilisation - NEVUS AFRICA Site (Cloud)

Ce guide vous explique comment gérer votre nouveau site NEVUS AFRICA hébergé sur le cloud.

## 🎯 Accès Admin

Pour gérer le contenu de votre site :
1.  Rendez-vous sur l'adresse de votre site suivie de `/admin` (ex: `https://votre-site.netlify.app/admin`).
2.  Connectez-vous avec vos identifiants :
    - **Utilisateur** : `admin`
    - **Mot de passe** : `root@2020`

---

## 🎨 Personnaliser l'apparence

Le nouveau design "Premium" est entièrement flexible.

1.  Connectez-vous au Back-office.
2.  Dans l'onglet **🎨 Apparence**, vous pouvez changer les couleurs institutionnelles :
    - **Couleur Primaire** (Bleu Marine par défaut)
    - **Couleur Secondaire**
    - **Couleur Accent** (Orange par défaut)
3.  Utilisez les sélecteurs de couleur et cliquez sur **Enregistrer les modifications**.

---

## 📝 Modifier le contenu des sections

1.  Allez dans l'onglet **📝 Contenu**.
2.  Vous pouvez modifier le titre et le texte de chaque section (Hero, À propos, Missions, Programmes).
3.  Cliquez sur **Mettre à jour** pour chaque section modifiée.

---

## 📧 Gérer les messages de contact

C'est une nouvelle fonctionnalité ajoutée pour vous :
1.  Allez dans l'onglet **📧 Messages**.
2.  Tous les messages envoyés via le formulaire de contact du site s'affichent ici.
3.  Vous pouvez lire les détails et supprimer les messages une fois traités.
4.  Une copie du message est également envoyée par email à `gloirebeniche11@gmail.com`.

---

## 📍 Coordonnées

1.  Allez dans l'onglet **📍 Contacts**.
2.  Mettez à jour votre email, numéro de téléphone et adresse physique.
3.  Cliquez sur **Sauver** pour chaque champ.

---

## ⚠️ Zone de Danger (Réinitialisation)

Si vous faites des erreurs ou si vous voulez repartir de zéro :
1.  Allez dans l'onglet **⚠️ Danger**.
2.  Cliquez sur **Réinitialiser tout le site**.
3.  Vous devrez taper `REINITIALISER` pour confirmer.
4.  Cela remettra toutes les couleurs et les textes à leur état d'origine (et supprimera les messages reçus).

---

## 🛠️ Maintenance technique (Supabase)

Votre site utilise **Supabase** comme base de données. 
- En cas de besoin technique, vous pouvez accéder à vos données sur le dashboard de [Supabase](https://supabase.com/).
- Ne partagez jamais vos clés API `SUPABASE_URL` et `SUPABASE_ANON_KEY`.

---

## 💡 Conseils

- 📱 Le site est optimisé pour les smartphones. Testez vos modifications sur mobile.
- 📸 Utilisez des images de haute qualité pour garder l'aspect "Premium".
- 📧 Vérifiez régulièrement l'onglet Messages pour ne rater aucune demande.

---

Bonne gestion de votre plateforme NEVUS AFRICA! 🌍❤️
